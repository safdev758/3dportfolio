# 🚀 3D Virtual Workspace

A modern, interactive 3D virtual developer workspace built with Three.js, featuring holographic screens, floating programming books, skill trophies, and an AI brain in a jar. This immersive environment showcases a futuristic developer's room with soft sci-fi aesthetics.

## ✨ Features

### 🖥️ **Holographic Screens**
- **React App Screen**: Displays live React code with syntax highlighting
- **Terminal Screen**: Shows development commands and git status
- **Analytics Dashboard**: Features colorful charts and system metrics
- All screens have glowing effects and subtle animations

### 📚 **Floating Book Library**
- **NestJS** - Backend framework expertise
- **React** - Frontend library mastery  
- **Docker** - Containerization skills
- **TypeScript** - Type-safe development
- **Node.js** - Server-side JavaScript
- Books gently float and rotate with individual animations

### 🏆 **Skill Trophy Shelf**
- **PostgreSQL** - Database management (Blue trophy)
- **MongoDB** - NoSQL database (Green trophy)
- **Prisma** - Database ORM (Dark trophy)
- **GraphQL** - API query language (Pink trophy)
- Each trophy rotates and glows with its technology's signature color

### 🧠 **AI Problem Solver**
- Brain-like AI floating in a glass jar labeled "Problem Solver"
- Pulsing pink/magenta glow effect
- Continuous rotation animation
- Represents AI-assisted development capabilities

### 🎨 **Visual Effects**
- **Cyan and purple gradient lighting** creating moody sci-fi atmosphere
- **Particle system** with floating colored particles
- **Dynamic lighting** with animated intensity changes
- **Soft shadows** and realistic material reflections
- **Glowing edges** on the desk and UI elements

## 🎮 Interactive Controls

### 🖱️ **Mouse Navigation**
- **Click + Drag**: Rotate camera around the workspace
- **Scroll Wheel**: Zoom in/out for detailed exploration
- **Smooth camera interpolation** for fluid movement

### 🎯 **Control Buttons**
- **Reset View**: Return camera to optimal viewing angle
- **Toggle Animation**: Pause/resume all animations
- **Switch Theme**: Toggle between cyberpunk and clean themes

## 🛠️ Technology Stack

- **Three.js** - 3D graphics and WebGL rendering
- **GSAP** - Smooth animations and transitions
- **Vite** - Fast development server and build tool
- **HTML5 Canvas** - Dynamic screen content generation
- **CSS3** - UI styling with glassmorphism effects

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser to `http://localhost:3000`

### Building for Production
```bash
npm run build
```
